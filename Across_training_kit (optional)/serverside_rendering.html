Server Side Rendering

You are free to use ajax or any other ways to parse server apis that returns json values

You can use jquery $.post to parse APIs


$.post("http://site.example.com/api/data").done(function(data){
	var n=JSON.parse(data);
	/* Json manipulation here*/
});

or 

$.ajax({
	url:"http://site.example.com/api/data",
	data:{topass:topass},
	method:"POST",
	success:function(data){ },
	error:function(err){ console.log(err); }

});

But.. you may also use vanilla javascript Ajax code and other more to render APIs
